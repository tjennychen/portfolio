# AI Builder Portfolio

Case studies and experiments in AI powered automation. Each project demonstrates a different application of AI tooling to solve real GTM, operations, and growth problems.

## Projects

- **I Hate Networking** (IHN): Chrome extension that automates conference networking via LinkedIn
- **KKlue GTM**: Go to market strategy and competitive intelligence for AI sales tools
- **Puzzle CRM**: Acupuncture clinic referral discovery and outreach automation
- **Rolling Damage Report**: Open mat finder for the BJJ community (1,987+ academies mapped)

## Structure

```
index.html            Landing page with hub and spoke navigation
ihn-portfolio/        I Hate Networking case study
kklue-portfolio/      KKlue GTM case study
```
